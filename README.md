# AEVUM — Premium Digital Agency Landing Page

A luxury, cinematic, and conversion-focused agency landing page built with pure HTML, CSS, and JavaScript.

---

## 🚀 Live Preview

> Deploy on GitHub Pages or Netlify for live preview.

---

## 🧱 Tech Stack

- **HTML5** — Semantic, clean structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No frameworks, no dependencies

---

## 🎨 Design System

- **Primary Font:** Syne (Display & Headings)
- **Secondary Font:** Manrope (Body)
- **Theme:** Dark luxury — charcoal base with cyan, violet & gold accents
- **All colors** managed via CSS variables in `:root`

---

```
aevum/
```

---

## 📄 Sections

1. **Header** — Sticky glass-blur navbar with CTA
2. **Hero** — Cinematic headline, stats, ambient orbs
3. **Features** — 6 premium feature cards
4. **About** — Split layout with visual frame
5. **Services** — 4 module service cards
6. **Testimonials** — 3 client quote cards
7. **CTA** — High-urgency call to action
8. **Footer** — Links, contact, credits

---

## ⚙️ JavaScript Features

- Sticky header on scroll
- Smooth scroll navigation
- Active nav link highlighting
- Scroll reveal animations
- Mobile hamburger menu
- Subtle cursor glow effect

---

## 🔘 Button Style Rule

All buttons follow this custom radius — right corners only:

```css
button {
  border-radius: 0 50px 50px 0;
}
```

No build tools required. Simply open in browser:

```bash
cd aevum
open index.html
```
